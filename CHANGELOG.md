## [0.1.0] - 2026-03-21

### Added
- Initial release
- Base58 encoding and decoding using the Bitcoin alphabet
- Base62 encoding and decoding for integers
- Base32 encoding and decoding using the Crockford variant
- Base85 encoding and decoding using the ASCII85 format
- Arbitrary base encoding and decoding for integers (base 2 to 62)
